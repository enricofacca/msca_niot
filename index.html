Network Inpainting via Optimal Transport

Project description

The Network Inpainting via Optimal Transport (NIOT) if a two-years project funded by the Marie-Curie action. This project has two main goals

  • define new mathematical and numerical tools for reconstruct of networks from corrupted images
  • applying these tools to reconstruct blood vessel networks in the human brain from corrupted MRI scan.
To pursue these goals, the central idea of the project is to incorporate novel mathematical tools provided by the Optimal Transport theory (in particular Branched Transport) into image processing techniques.

This article summarize our achiviement toward our first goal.
